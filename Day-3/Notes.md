## What is DOM?
- (Document Object Model)
- It is a hierarchy of elements present in browser.
- Chrome : Blink, V8

## Workflow of Browser Engines
Markup -> Bytes -> Chars -> Token -> nodes -> DOM -> Layout -> Rendering -> Painting

Styles -> Bytes -> Chars -> CSSOM -> Layout -> Rendering -> Painting


## What is Shadow DOM?
Shadow DOM is a DOM under any specific element in a page

<input type = "file">
-> <input type = "text">
-> <button>Browse</button>


## What is Virtual DOM?
- The virtual DOM is a programming concept where an ideal, or "VIRTUAL", representation of a UI is kept in memory and synced with the "REAL" DOM by a library such as ReactDOM. this process is known as **RECONCILIATION**
