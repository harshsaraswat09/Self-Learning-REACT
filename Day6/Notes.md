## React Components

- Component is a template used for building application
- Component are building Block for React application
- Every component comprises
    a. Markup
    b. styles
    c. Logic
- Markup is designed using HTML
- Styles are defined using CSs, Tailwind CSS, Bootstrap
- JSX [JavaScript Extension]
- JavaScript or TypeScript(tsx)

- Components allows resuability and easy extensibility
- You can Build custom components or you can use existing component provided by various 3rd party

-----------------------------------

## Building a React Component

- React Component is a function or class of JS that returns markup, styles, and configures functionality.

- Component function or class are defined in simple js file 'Component.js' [modules]

- Component is function or class defined in a module

- In order to use function or class from module you chave to mark it as 
"export"
    - export function name(){}
    - export class name{}

- you can also configure "default export"

- you have to import modules and components

    - import { Name } from "Module"
    (if it is not default)
    - import { Name } from "Module"
    (if it is default)

- Every component must return markup by using "return" keyword
