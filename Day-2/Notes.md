## Difference between Library and Framework?

Web Application 
-> library 
-> page Loads into browser 
-> Loads Static DOM 
-> Load Library
-> Library is Passive 
-> browser uses Event to make the library Active
-> Static DOM is converted into Dynamic DOM


Web Application
-> Framework
-> page Loads into browser 
-> Loads Static DOM 
-> Load Framework
-> Framework is active
-> Static DOM is converted into Dynamic DOM


- library can build applications
- Frameworks can build and control the application flow

## What is React?
- React is a Javascript library for building UI
- UI = SPA
- Fluid 
- Unified
- Loosly Coupled and extensible


## What are the features of React?
Key Features
1. Component based
    - Easy to build interfaces
    - Easy to reuse
    - Building faster
    - Easy to extend

2. Modular Library 
    - Application Specific library
    - improves the performance of application
    - light weight application

3. Asynchronous Library
    - Implicitly uses Ajax
    - without reloading the complete page
    new details are added to page

4. uses Virtual DOM
